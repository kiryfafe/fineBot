# Telegram Bot для игры "Быки и Коровы"

Этот бот позволяет играть в классическую игру "Быки и Коровы" через Telegram.

## 📋 Описание

Бот загадывает число с уникальными цифрами, а игрок должен его угадать. После каждой попытки бот сообщает:
- 🐂 **Быки** — цифры на правильных местах
- 🐄 **Коровы** — правильные цифры не на своих местах

## 🚀 Быстрый старт

### 1. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 2. Получение токена бота

1. Откройте Telegram и найдите [@BotFather](https://t.me/BotFather)
2. Отправьте команду `/newbot`
3. Следуйте инструкциям:
   - Придумайте имя для вашего бота (например, "Bulls and Cows Game")
   - Придумайте username для бота (должен заканчиваться на `bot`, например `bulls_cows_game_bot`)
4. BotFather выдаст вам токен (выглядит как `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`)

### 3. Запуск бота

```bash
python telegram_bot.py YOUR_BOT_TOKEN
```

Замените `YOUR_BOT_TOKEN` на токен, полученный от BotFather.

## 🎮 Как играть

### Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветственное сообщение и правила игры |
| `/newgame [3-6]` | Начать новую игру (можно указать длину числа: 3, 4, 5 или 6 цифр) |
| `/help` | Показать справку по игре |
| `/stats` | Показать статистику текущей игры |
| `/cancel` | Завершить текущую игру |

### Пример игры

```
Вы: /newgame
Бот: 🎮 Новая игра началась! Я загадал число из 4 цифр.

Вы: 1234
Бот: 🐂 1 бык | 🐄 2 коровы

Вы: 5678
Бот: 🐂 0 быков | 🐄 1 корова

Вы: 1594
Бот: 🎉 Поздравляю! Ты выиграл! Загаданное число: 1594
```

Или другой пример:
```
Загадано: 5189
Вы: 5281
Бот: 🐂 2 быка | 🐄 1 корова (5 и 8 на правильных местах, 1 есть, но не там)
```

## 🔧 Настройки

### Длина числа

По умолчанию бот загадывает число из 4 цифр. Вы можете изменить это, указав длину числа при создании новой игры:

```
/newgame 3  # число из 3 цифр
/newgame 5  # число из 5 цифр
/newgame 6  # число из 6 цифр
```

## 🛠 Технические детали

- **Язык**: Python 3.7+
- **Библиотека**: python-telegram-bot v21.0
- **Хранение состояния**: В памяти (сбрасывается при перезапуске бота)

## ⚠️ Важные замечания

1. **Хранение данных**: Текущая версия хранит состояние игр только в памяти. При перезапуске бота все активные игры будут потеряны.

2. **Правила игры**:
   - Число не может начинаться с 0
   - Все цифры в числе должны быть уникальными
   - Длина числа: от 3 до 6 цифр

3. **Валидация ввода**: Бот проверяет:
   - Что введены только цифры
   - Что длина числа соответствует настройкам игры
   - Что нет повторяющихся цифр
